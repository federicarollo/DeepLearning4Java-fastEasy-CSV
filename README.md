## Build and Run

Use [Maven](https://maven.apache.org/) to build the examples. 

```
mvn clean package
```
```
(1) mvn install

(2) java -cp dl4j-examples/target/dl4j-examples-0.4-rc0-SNAPSHOT-bin.jar "org.deeplearning4j.examples.dataExamples.CSVExample" XOR.csv 2 6 250 .1 1000
```
